Using Psucculint

  1. Make sure you have a minimum of Java JDK 17 installed on your computer. You can find it here.
  2. Download the latest version of Psucculint here.
  3. Unzip the Psucculint download to your desired location (preferably close to your pseudocode path).

    • Run the bat file (if your firewall tries to block it just bypass the warning).
      •  or
    • Run the jar file with the command line. (you may pass in your pseudocode path as the first argument if you wish)
  5. Input the path to your pseudocode docx or txt file. Ex: "folder1/folder2/myPseudo.docx"
  6. Review your results in the command line or in the "<YourFileName>PsucculintResults.txt" file generated.
